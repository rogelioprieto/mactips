# How to install OBS Studio

## Problem
I need to install a software to edit my background in videos in videoconference software (zoom, google meet, etc).

## Solution
Install OBS Studio and a virtual webcam.

### Step by step

#### 1. Install OBS studio (Windows or mac). Download and install from official website.\
<https://obsproject.com/>

#### 2. Install VirtualCam.

   * 2.1 Windows\
<https://github.com/Fenrirthviti/obs-virtual-cam/releases>

   * 2.2 Mac\
Open a ```Terminal``` and type this command
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"
```
   Download e install this:\
<https://github.com/johnboiles/obs-mac-virtualcam/releases/download/v1.2.0/obs-mac-virtualcam-3ca8f62-v1.2.0.pkg>

#### 3. Install VB- Cable Virtual Audio (Windows y Mac)\
<https://www.vb-audio.com/Cable/index.htm>
